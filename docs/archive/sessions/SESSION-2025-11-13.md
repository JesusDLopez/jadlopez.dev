## Key Learnings

### CSS Variable Hierarchy
CSS variables in `:root` can override direct CSS properties when used in `calc()` functions. Always check custom properties when height values don't respond to changes.

### Absolute Positioning & Parent Height
Absolutely positioned elements require parent containers with defined height. When parent is `height: auto` but child is `height: 100vh`, overflow occurs.

### Z-Index Stacking Contexts
For z-index to work, element must have `position: relative/absolute/fixed`. Even if one section has `z-index: 1`, sections without explicit z-index can end up behind it visually.

### Text Rendering with Gradients
Complex nested spans with `background-clip: text` can cause text breaking issues. Simpler structure with single element and `<br />` tags is more reliable.

### Footer Positioning
Setting `padding: 0` on bottom with individual padding on last child element allows precise control of footer edge spacing.
